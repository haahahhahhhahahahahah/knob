# Dark Control Widget

Pixel-focused React recreation of a dark skeuomorphic brightness/volume control.

## Run

```bash
npm install
npm run dev
```

## Screenshot States

```bash
npm run test:screenshots
```

Screenshots are written to `test-results/widget-states/`:

- `brightness-52.png`
- `brightness-low-transition.png`
- `volume-53.png`
- `volume-72.png`
- `volume-14.png`
- `volume-0.png`
